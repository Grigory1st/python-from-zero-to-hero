# Week 4: Data Structures

## Theory

Data structures store and organize data efficiently.

### Lists
Mutable sequences.

```python
fruits = ["apple", "banana"]
fruits.append("cherry")
print(fruits[0])  # apple
```

### Tuples
Immutable sequences.

```python
point = (1, 2)
```

### Dictionaries
Key-value pairs.

```python
person = {"name": "Alice", "age": 25}
print(person["name"])
```

### Sets
Unique elements.

```python
nums = {1, 2, 3}
nums.add(4)
```

### List Comprehensions
```python
squares = [x**2 for x in range(5)]
```

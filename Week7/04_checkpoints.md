# Week 7: Checkpoints

1. Handle exceptions? (try, catch, except, error)

<details>
<summary>🇷🇺 Обработать исключения? (try, catch, except, error)</summary>

Ответ: `except`. В Python используется ключевое слово `except`, а не `catch`.

</details>

2. Raise exception? (raise, throw, error, except)

<details>
<summary>🇷🇺 Вызывать исключение? (raise, throw, error, except)</summary>

Ответ: `raise`. Ключевое слово `raise` генерирует исключения.

</details>

3. Always executes? (finally, else, except, try)

<details>
<summary>🇷🇺 Луқсэ всегда выполняется? (finally, else, except, try)</summary>

Ответ: `finally`. Блок `finally` эксекутируется внезависимо от того, оказалась ли ошибка.

</details>

4. Debug tool? (pdb, debug, print, log)

<details>
<summary>🇷🇺 Инструмент отладки? (pdb, debug, print, log)</summary>

Ответ: `pdb`. Пдб - Python DeBugger, официальный дебюгер Python.

</details>

5. Custom exception? (class Exception, def exception, raise Exception, except Exception)

<details>
<summary>🇷🇺 Пользовательское исключение? (class Exception, def exception, raise Exception, except Exception)</summary>

Ответ: `class Exception`. Определите новые исключения, наследуя от `Exception`.

</details>

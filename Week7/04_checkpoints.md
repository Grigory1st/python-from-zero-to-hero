# Week 7: Checkpoints

1. Handle exceptions? (try, catch, except, error)

<details>
<summary>ru</summary>

Ответ: `except`. В Python используется ключевое слово `except`, а не `catch`.

</details>

2. Raise exception? (raise, throw, error, except)

<details>
<summary>ru</summary>

Ответ: `raise`. Ключевое слово `raise` генерирует исключения.

</details>

3. Always executes? (finally, else, except, try)

<details>
<summary>ru</summary>

Ответ: `finally`. Блок `finally` эксекутируется внезависимо от того, оказалась ли ошибка.

</details>

4. Debug tool? (pdb, debug, print, log)

<details>
<summary>ru</summary>

Ответ: `pdb`. Пдб - Python DeBugger, официальный дебюгер Python.

</details>

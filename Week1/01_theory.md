# Week 1: Introduction to Python - Basics

## Theory

Welcome to your journey from zero to junior Python developer! This week, we'll cover the fundamentals of Python programming, focusing on variables, data types, and basic operations. These concepts are the building blocks for writing scripts that automate real-world tasks, like calculating expenses or processing user input.

### What is Python?
Python is a high-level, interpreted programming language known for its simplicity and readability. It's widely used in web development, data analysis, automation, and more. You'll write code in plain English-like syntax, making it easier to learn and maintain.

### Variables
Variables are containers for storing data values. In Python, you don't need to declare the type; it's inferred automatically.

```python
# Example: Storing user information
name = "John Doe"
age = 30
is_employee = True
```

### Data Types
Python has several built-in data types:

- **int**: Whole numbers (e.g., 42)
- **float**: Decimal numbers (e.g., 3.14)
- **str**: Text strings (e.g., "Hello, World!")
- **bool**: True or False values

```python
# Examples
temperature = 25.5  # float
message = "Welcome to Python!"  # str
is_active = False  # bool
```

### Basic Operations
You can perform arithmetic operations on numbers and concatenate strings.

```python
# Arithmetic
a = 10
b = 3
sum_result = a + b  # 13
difference = a - b  # 7
product = a * b  # 30
quotient = a / b  # 3.333...

# String operations
greeting = "Hello"
name = "Alice"
full_greeting = greeting + " " + name  # "Hello Alice"
```

### Input and Output
To interact with users, use `input()` for input and `print()` for output.

```python
# Getting user input
user_name = input("Enter your name: ")
print("Hello, " + user_name + "!")
```

### Comments
Comments are notes in your code that Python ignores. Use `#` for single-line comments.

```python
# This is a comment
print("This will run")  # Inline comment
```

By the end of this week, you'll be able to write simple scripts for tasks like greeting users or basic calculations.

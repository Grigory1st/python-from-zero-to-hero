### What is Python?

Python is a high-level, interpreted programming language known for its simplicity and readability. It's widely used in web development, data analysis, automation, and more. You'll write code in plain English-like syntax, making it easier to learn and maintain.

<details>
<summary>ru</summary>

Python — это высокоуровневый интерпретируемый язык программирования, известный своей простотой и читаемостью. Он широко используется в веб-разработке, анализе данных, автоматизации и многом другом. Вы будете писать код в синтаксисе, похожем на обычный английский, что облегчает обучение и поддержку.

</details>

### Variables

Variables are containers for storing data values. In Python, you don't need to declare the type; it's inferred automatically.

<details>
<summary>ru</summary>

Переменные — это контейнеры для хранения значений данных. В Python вам не нужно объявлять тип; он определяется автоматически.

</details>

```python
# Example: Storing user information
name = "John Doe"
age = 30
is_employee = True
```

<details>
<summary>ru</summary>

```python
# Пример: Хранение информации о пользователе

```

</details>

### Data Types

Python has several built-in data types:

<details>
<summary>ru</summary>

В Python есть несколько встроенных типов данных:

</details>

- **int**: Whole numbers (e.g., 42)
- **float**: Decimal numbers (e.g., 3.14)
- **str**: Text strings (e.g., "Hello, World!")
- **bool**: True or False values

<details>
<summary>ru</summary>

- **int**: Целые числа (например, 42)
- **float**: Десятичные числа (например, 3.14)
- **str**: Текстовые строки (например, "Hello, World!")
- **bool**: Значения True или False

</details>

```python
# Examples
temperature = 25.5  # float
message = "Welcome to Python!"  # str
is_active = False  # bool
```

### Basic Operations

You can perform arithmetic operations on numbers and concatenate strings.

<details>
<summary>ru</summary>

Вы можете выполнять арифметические операции с числами и объединять строки.

</details>

```python
# Arithmetic
a = 10
b = 3
sum_result = a + b  # 13
difference = a - b  # 7
product = a * b  # 30
quotient = a / b  # 3.333...

# String operations
greeting = "Hello"
name = "Alice"
full_greeting = greeting + " " + name  # "Hello Alice"
```

<details>

### Input and Output

To interact with users, use `input()` for input and `print()` for output.

<details>
<summary>ru</summary>

Для взаимодействия с пользователями используйте `input()` для ввода и `print()` для вывода.

</details>

```python
# Getting user input
user_name = input("Enter your name: ")
print("Hello, " + user_name + "!")
```

<details>
<summary>ru</summary>


### Comments

Comments are notes in your code that Python ignores. Use `#` for single-line comments.

<details>
<summary>ru</summary>

Комментарии — это заметки в вашем коде, которые Python игнорирует. Используйте `#` для однострочных комментариев.

</details>

```python
# This is a comment
print("This will run")  # Inline comment
```

<details>
<summary>ru</summary>

```python
# Это комментарий
print("This will run")  # Встроенный комментарий
```

</details>
